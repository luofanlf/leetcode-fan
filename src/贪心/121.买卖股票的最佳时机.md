# 121.买卖股票的最佳时机

https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    public int maxProfit(int[] prices) {
        int minPrice = prices[0];
        int maxProfit = 0;
        for(int i = 1;i<prices.length;i++){
            if(prices[i]<minPrice){
                minPrice = prices[i];
            }else{
                int profit = prices[i]-minPrice;
                maxProfit = Math.max(profit,maxProfit);
            }
        }
        return maxProfit;
    }
}
```

贪心算法，假设可以以最低的价格买入，最高的价格买出。声明一个变量来存储最低的买入价格一个变量来存储最大利润，在一次遍历中，如果当前价格小于最低买入价格则更新最低买入价格；如果高于最低买入价格，则用当前的价格来减去最低买入价格来计算利润：与当前最大利润对比并决定是否更新

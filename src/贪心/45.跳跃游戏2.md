# 45.跳跃游戏

https://leetcode.cn/problems/jump-game-ii/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    public int jump(int[] nums) {
        int length = nums.length-1;
        int maxDis = 0;
        int currEnd = 0;
        int res = 0;
        for(int i = 0;i<length;i++){
            maxDis = Math.max(i+nums[i],maxDis);
            if(i == currEnd){
                res++;
                currEnd = maxDis;
            }
        }
        return res;
    }
}
```

贪心算法，从下标为0处出发，更新能从0出发到达的最远位置并且将最远位置设置为currEnd，然后开始遍历，在到达currEnd之前

，一边遍历一边更新能达到的最远位置。当达到当前的currEnd后，更新currEnd为当前能到达的最远距离继续遍历

# 55.跳跃游戏

https://leetcode.cn/problems/jump-game/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    public boolean canJump(int[] nums) {
        int maxDis = 0;
        for(int i = 0;i<nums.length;i++){
            if(i<=maxDis){
                maxDis = Math.max(maxDis,i+nums[i]);
            }
        }
        if(maxDis<nums.length-1){
            return false;
        }
        return true;
    }
}
```

贪心算法，对于每个点，能跳跃到的最远位置是i+nums[i]，因此只需要一次遍历，设置初始最远距离为0，在当前位置小于最远跳跃距离即可以被达到时，更新maxDis，取``Math.max(maxDis,i+nums[i])``。在遍历完成后，如果最远跳跃距离小于最后一个元素则返回false

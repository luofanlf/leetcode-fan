# [64. 最小路径和](https://leetcode.cn/problems/minimum-path-sum/)

```java
class Solution {
    /**
        dp[i][j]-  机器人达到i，j这个位置可能的路径数
        dp[i][j] = dp[i][j-1]+dp[i-1][j]
     */
    public int uniquePaths(int m, int n) {
        int[][] dp = new int[m][n];
        for(int i = 0;i<m;i++){
            dp[i][0] = 1;
        }
        for(int i = 0;i<n;i++){
            dp[0][i] = 1;
        }
        for(int i = 1;i<m;i++){
            for(int j = 1;j<n;j++){
                dp[i][j] = dp[i-1][j] + dp[i][j-1];
            }
        }
        return dp[m-1][n-1];
    }
}
```

动态规划，对于dp数组定义和动态转移方程和[62. 不同路径](https://leetcode.cn/problems/unique-paths/)类似

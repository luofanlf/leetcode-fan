# 70.爬楼梯

https://leetcode.cn/problems/climbing-stairs/description/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    /** 
        dp[n]- 对于n阶的台阶有dp[n]种方法可以到达 
        dp[n] = d[n-1]+d[n-2]
    **/
    
    public int climbStairs(int n) {
        int[] dp = new int[n+1];
        dp[0] = 1;
        dp[1] = 1;
        for(int i =2;i<n+1;i++){
            dp[i] = dp[i-1]+dp[i-2];
        }
        return dp[n];
    }
}
```

动态规划，数组dp[n]代表对于n阶的台阶有dp[n]种方法可以到达 ，因为每次可以爬一阶或者两阶，所以对于dp[n]有状态转移方程：dp[n]=dp[n-1]+dp[n-2]。例如爬三节楼梯，可以理解为所有爬一节的方案然后再爬两节，所有爬两节的方案在爬一节。一个for循环得出所有的解，返回dp[n]。

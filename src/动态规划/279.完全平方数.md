# 279.完全平方数

https://leetcode.cn/problems/perfect-squares/description/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    /**
        dp[n] - 和为n的完全平方数的最少数量
        状态转移方程 - dp[n] = Math.min(dp[n-k^2]+1)
     */
    public int numSquares(int n) {
        int[] dp = new int[n+1];
        dp[0] = 0;
        for(int i = 1;i<=n;i++){
            dp[i] = Integer.MAX_VALUE;
            for(int j = 1;j*j<=i;j++){
                dp[i] = Math.min(dp[i],dp[i-j*j]+1);
            }
        }
        return dp[n];
    }
}
```

动态规划：dp[n] - 和为n的完全平方数的最少数量； 状态转移方程 - dp[n] = Math.min(dp[n-k^2]+1)，在k^2小于n时，dp[n]等于dp[n-k^2]+1，因为只需要加上一个数也就是k的平方，因此从1到n推导dp[n]并对每个值的k进行遍历

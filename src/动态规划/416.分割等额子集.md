# 416.分割等额子集

https://leetcode.cn/problems/partition-equal-subset-sum/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    /**
     dp[n] - 能否用数组中的数组成sum n
     dp[n] = dp[n] || dp[n-v]
     */
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for(int i = 0;i<nums.length;i++){
            sum+=nums[i];
        }
        if(sum % 2 != 0){
            return false;
        }
        boolean[] dp = new boolean[sum+1];
        dp[0] = true;
        for(int v : nums){
            for(int j = sum;j>=v;j--){
                dp[j] = dp[j] || dp[j-v];
            }
        }
        return dp[sum/2];
    }
}
```

先取数组sum，如果无法被2整除直接返回false，如果可以，问题就转化为了和为sum/2的01背包问题

定义dp[n]为能否用数组中的数组成n，转移方程dp[n] = dp[n] || dp[n-v],v为数组中的数，对于状态转移方程： dp[j] = dp[j] || dp[j-v];对于每个v有选与不选的选择，只要其中一个为true则dp[j]为true；另外因为每个数只可以被使用一次，因此倒序遍历更新dp，例如数组中有2，对于dp[4]，在有dp[4] = dp[4] || dp[4-2]的时候因为dp[2]还没有被更新因此不会出现重复使用2的情况

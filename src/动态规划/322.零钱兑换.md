# 322.零钱兑换

https://leetcode.cn/problems/coin-change/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    /**
        dp[n] - 组成amount n所需的最少硬币
        状态转移方程 - dp[n] = Math.min(dp[n-coins[k]]+1) k为coins中的选择
     */
    public int coinChange(int[] coins, int amount) {
        int[] dp = new int[amount+1];
        dp[0] = 0;
        for(int i = 1;i<=amount;i++){
            dp[i] = Integer.MAX_VALUE;
            for(int j = 0;j<coins.length;j++){
                if(coins[j]<=i && dp[i-coins[j]]!= -1 ){
                    dp[i] = Math.min(dp[i],dp[i-coins[j]]+1);
                }
            }
            if(dp[i]== Integer.MAX_VALUE){
                dp[i] = -1;
            }
        }
        return dp[amount];
    }
}
```

动态规划，遍历所有的选择，对当前的dp[n]为dp[n-coins[k]]+1中最小的值，把无法到达的值更新为-1，只有在``coins[j]<=i && dp[i-coins[j]]!= -1``时进行更新（如果硬币大于当前面额则没有必要更新，如果没有硬币组合满足i-coins[j]则没有必要更新），在更新前把初始值设置为Integer.MAX_VALUE，如果遍历了所有硬币选择后还为Integer.MAX_VALUE说明没有满足要求的组合，更新为-1。

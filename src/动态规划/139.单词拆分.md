# 139.单词拆分

https://leetcode.cn/problems/word-break/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
/**
    dp[i] - 对于字符串s的前i个字符能否用wordDict中的单词组成
    dp[i] = dp[j]&& dp[j..i-1]
 */
    public boolean wordBreak(String s, List<String> wordDict) {
        boolean[] dp = new boolean[s.length()+1];
        Set<String> set = new HashSet<>(wordDict);
        dp[0] = true;
        for(int i = 1;i<=s.length();i++){
            for(int j = 0;j<i;j++){
                if(dp[j]&&set.contains(s.substring(j,i))){
                    dp[i]=true;
                    break;
                }
            }
        }
        return dp[s.length()];
    }
}
```

因为是判断s是否合法，所以dp[i]定义为字符串s的前i个字符是否可以用wordDict中的单词组成。所以对于前i个字符，只需要遍历分割点j并且保证dp[j]为true和s[j..i]在wordDict中即可说明dp[i]为true。判断s[j..i]是否在wordDict中可以把wordDict用set存储然后用substring(j,i)进行判断。因此两个循环，外层遍历i构建dp数组，内层遍历分割点j进行判断。

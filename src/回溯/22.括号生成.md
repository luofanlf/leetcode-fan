# 22.括号生成

https://leetcode.cn/problems/generate-parentheses/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    private List<String> res = new ArrayList<>();

    private char[] choices = new char[]{'(',')'};

    public List<String> generateParenthesis(int n) {
        StringBuilder state = new StringBuilder();
        dfs(state,0,0,n);
        return res;
    }
    public void dfs(StringBuilder state, int leftNum, int rightNum, int n){
        if(leftNum == n && rightNum == n){
            res.add(state.toString());
            return;
        }
        for(char c:choices){
            if(leftNum<=n&&rightNum<=n&&leftNum>=rightNum){
                state.append(c);
                if (c == '(') {
                    leftNum++;
                } else {
                    rightNum++;
                }
                
                dfs(state,leftNum,rightNum,n);

                //回溯
                state.deleteCharAt(state.length() - 1);
                if (c == '(') {
                    leftNum--;
                } else {
                    rightNum--;
                }   

            }
        }
    }
}
```

回溯，在n对括号的情况下，要保证括号有效，只需要记录左括号数量，右括号数量，在每次选择括号的时候只需要保证左右括号数量都<n并且左扩号的数量大于等于右括号就能保证括号有效（如果左扩号数量小于右括号则有右括号无法正确闭合）

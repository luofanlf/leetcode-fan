# 46.全排列

https://leetcode.cn/problems/permutations/description/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    public List<List<Integer>> permute(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        List<Integer> state = new ArrayList<>();
        boolean[] used = new boolean[nums.length];
        backTrack(state,nums,used,res);
        return res;
    }

    public void backTrack(List<Integer> state,int[] choice,boolean[] used,List<List<Integer>> res){
        //判断是否为解
        if(state.size() == choice.length){
            //记录解
            res.add(new ArrayList<>(state));
            return;
        }
        //遍历所有选择
        for(int i = 0;i<choice.length;i++){
            //做出选择，更新state和choice
            if(used[i]){
                continue;
            }
            state.add(choice[i]);
            used[i] = true;

            //继续搜索
            backTrack(state,choice,used,res);

            //回溯
            state.remove(state.size() - 1);
            used[i] = false;
            
        }
    } 
}
```

回溯，建立一个boolean数组used记录当前数字是否可以被使用与数组nums共同组成当前的choice，当state长度达到nums长度时说明排列完成得到solution，return来向上回溯，在回溯的时候移除state中新添加的元素并且在choice将该元素标记为未使用。

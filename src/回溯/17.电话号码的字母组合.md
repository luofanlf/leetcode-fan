# 17.电话号码的字母组合

https://leetcode.cn/problems/letter-combinations-of-a-phone-number/description/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    private List<String> result = new ArrayList<>();

    public List<String> letterCombinations(String digits) {
        //映射到一个string列表中
        String[] map = {"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};
        List<String> choice = new ArrayList<>();
        for(char c: digits.toCharArray()){
            int digit = c -'0';
            choice.add(map[digit]);
        }

        char[] state = new char[digits.length()];
        dfs(state,choice,0);
        return result;
    }
    public void dfs(char[] state,List<String> choice,int start){

        if(start==choice.size()){
            result.add(new String(state));
            return;
        }
        
        //当前数字对应的字符串
        String curr = choice.get(start);

        for(int i = 0;i<curr.length();i++){
            state[start] = curr.charAt(i);
            //遍历下一个数字对应字母
            dfs(state,choice,start+1);
            //不需要手动回溯因为state数组会被自动覆盖
        }
    
    }
}
```

回溯，首先把电话号码每个数字对应的字母映射到一个字符串链表中，然后开始dfs，start代表处理的是第几个数字，用一个char数组来记录当前的state，这个数组的长度等于数字的长度，首先从列表中得到start对应的字符串curr，然后遍历curr中的每个字母，然后递归dfs处理下一个数字，因为我们采用的数组来存储状态，所以无需手动恢复状态，在回溯后递归过程中会被自动覆盖

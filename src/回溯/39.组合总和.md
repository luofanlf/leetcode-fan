# 39.组合总和

https://leetcode.cn/problems/combination-sum/description/?envType=study-plan-v2&envId=top-100-liked

```java
class Solution {
    private List<List<Integer>> res = new ArrayList();
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        Arrays.sort(candidates);
        List<Integer> state = new ArrayList<>();
        dfs(state,0,candidates,target,0);
        return res;
    }

    public void dfs(List<Integer> state,int curSum,int[] choices,int target,int start){
        if(curSum == target){
            res.add(new ArrayList(state));
            return;
        }
        if(curSum>target){
            return;
        }

        for(int i = start;i<choices.length;i++){
            if(curSum+choices[i]>target)
                break;
            state.add(choices[i]);
            curSum += choices[i];

            dfs(state,curSum,choices,target,i);

            //回溯
            state.remove(state.size()-1);
            curSum-=choices[i];
        }
    }
}
```

用回溯的思路，一个list和一个int记录当前的选择和当前选择元素的总和。为了避免重复解，将choice排序然后记录一个start来让在choices里面选择解的时候只能从当前元素往后找，就不会出现[1,2],[2,1]这样的重复解。

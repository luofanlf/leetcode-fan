# 27.移除元素

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int fastIndex = 0;
        int slowIndex = 0;
        for(int i =0 ;i<nums.length;i++){
            if(nums[i]!=val){
                nums[slowIndex]=nums[fastIndex];
                slowIndex++;
            }
            fastIndex++;
        }
        return slowIndex;
    }
}
```

**双指针法**，快指针一直更新，慢指针遇到要删除的值的时候不更新，用快指针遍历到不等于val的值来填进慢指针的index里

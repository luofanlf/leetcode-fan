# 数组

### define an Array

* 声明+指定长度:     ``int[] arr = new int[5];``

* 声明+初始化： ``int[] arr = {1, 2, 3, 4, 5};``

* 使用new初始化

  ```java
  int[] arr = new int[] {1, 2, 3, 4, 5};
  String[] names = new String[] {"Alice", "Bob"};
  ```

### get the length of array

```java
int[] arr = new int[] {1, 2, 3, 4, 5};
String[] names = new String[] {"Alice", "Bob"};
```

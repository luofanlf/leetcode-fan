# [169. 多数元素](https://leetcode.cn/problems/majority-element/)

```java
class Solution {
    public int majorityElement(int[] nums) {
        int count = 1;
        int candidate = nums[0];
        for(int i = 1;i<nums.length;i++){
            if(count==0){
                candidate = nums[i];
                count = 1;
            }else{
                if(nums[i]==candidate){
                    count++;
                }else{
                    count--;
                }
            }
        }
        return candidate;
    }
}
```

声明两个int，一个记录candidate一个记录count，初始化为nums[0]和1，遍历nums，如果当前num不等于candidate，就使count减1，如果等于则+1，如果count被减到0了，在遍历的下一个num更新其为candidate和count=1；最后留下的candidate即为多数元素

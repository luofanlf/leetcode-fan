# [75. 颜色分类](https://leetcode.cn/problems/sort-colors/)

```java
class Solution {
    public void sortColors(int[] nums) {
        int left = 0;
        int right = nums.length-1;
        int mid = 0;
        while(mid<=right){
            if(nums[mid]==0){
                int temp = nums[left];
                nums[left] = 0;
                nums[mid] = temp;
                mid++;
                left++;
            }else if(nums[mid]==1){
                mid++;
            }else{
                int temp = nums[right];
                nums[right] = 2;
                nums[mid] = temp;
                right--;
            }
        }
    }
}
```

对于值域小的问题，我们可以循环统计0，1，2分别出现的次数然后填入，但这样的话是两次遍历，我们需要寻找能够边遍历边填入的做法。因此我们可以维护三个区间[0,left-1],[left..mid..right],[right+1..n-1];这三个区域分别用于存放0,1,2。也就是说left左边是确定的0，right右边都是确定的2，left-mid是确定的1，mid-right是不确定区域，因此循环条件mid<=right。当mid=0时，把mid与left位置的数交换，left位置一定是1，因此换过来的一定是确定的，因此把mid和left都+1；当mid=2时，交换mid和right的位置，扩大right的边界，但是不知道换过来的是什么，因此mid不动继续处理当前的新换过来的mid。

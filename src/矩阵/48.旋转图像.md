# [48. 旋转图像](https://leetcode.cn/problems/rotate-image/)

```java
class Solution {
    public void rotate(int[][] matrix) {
        //矩阵转置+行反转
        int n = matrix.length;

        //转置
        for(int i = 0;i<n;i++){
            for(int j = 0;j<=i;j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }

        //行反转
        for(int i = 0;i<n;i++){
            int left = 0;
            int right = n-1;
            while(left<right){
                int temp = matrix[i][left];
                matrix[i][left] = matrix[i][right];
                matrix[i][right] = temp;
                left++;
                right--;
            }
        }
    }
}
```

顺时针旋转九十度对于矩阵中的(i,j)来说，坐标转化成了(j,n-1-i)其实就是矩阵转置再加上行反转的效果，那么先沿主对角线矩阵转置，交换``matrix[i][j]和matrix[j][i]``，然后行反转，遍历每一行交换``matrix[i][left]和matrix[j][right]``

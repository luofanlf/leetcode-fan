# [73. 矩阵置零](https://leetcode.cn/problems/set-matrix-zeroes/)

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean[] zeroRow = new boolean[m];
        boolean[] zeroCol = new boolean[n];

        for(int i = 0;i<m;i++){
            for(int j = 0;j<n;j++){
                if(matrix[i][j]==0){
                    zeroRow[i] = true;
                    zeroCol[j] = true;
                }
            }
        }

        for(int i = 0;i<m;i++){
           for(int j = 0;j<n;j++){
                if(zeroRow[i]|zeroCol[j]){
                    matrix[i][j]= 0;
                }
           }
        }
    }
}
```

建立两个额外的数组用来记录改行/列是否有0存在，两遍遍历先检验0的存在情况第二遍更新数组，O(m+n)的额外空间，可以优化至O(1)的额外空间，用第一行和第一列来记录是否有0存在，然后第一行第一列原本是否有0存在用一个额外的变量记录。

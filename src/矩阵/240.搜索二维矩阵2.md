# [240. 搜索二维矩阵 II](https://leetcode.cn/problems/search-a-2d-matrix-ii/)

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length;
        int n = matrix[0].length;

        int i = 0;
        int j = n-1;
        while(i<m && j>=0){
            if(matrix[i][j]>target){
                j--;
            }
            else if(matrix[i][j]<target){
                i++;
            }
            else{
                return true;
            }
        }
        return false;
    }
}
```

因为行，列都是递增的，所以我们可以利用这个性质从右上角``matrix[0][n-1]``出发，对于这个点，他左边的所有数都比他小，下面的所有数都比他大，因此我们将这个数与target比较，如果这个数大于target，那么我们就可以排除掉n-1这一列所有比它大的数字，然后向左移动；如果这个数小于target就可以排除掉这个数左边的所有数字然后往下移动；在越界前一直重复这个操作，如果找到了target就返回true；越界了就结束循环说明不存在target返回false；
